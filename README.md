# **Passo de execussão do sistema**

Abaixo segue um passo a passo para iniciar a apicação

---

## **Flask**

- cd \demo\flaskr\
  -python -m venv .venv
- venv\Scripts\activate
- pip install -r requirements.txt --upgrade

- flask --app flaskr run --debug

## **React**

- cd \Front\
  -npm install
- npm run dev
